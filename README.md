# Chess Landing

Адаптивный лендинг, сверстанный по макету из Figma в рамках тестового задания.

## Стек

- HTML
- CSS
- JavaScript

## Что реализовано

- pixel-perfect верстка по макету
- адаптация под desktop и mobile
- бегущая строка в `header` и `footer`
- слайдер этапов без autoplay и без loop
- зацикленная карусель участников с autoplay
- якорные кнопки на первом экране
- плавные reveal-анимации интерфейса

## Структура проекта

```text
src/
  assets/
  scripts/
  styles/
  index.html
```

## Локальный запуск

Проект статический, поэтому его можно открыть любым локальным сервером.

Например, через VS Code Live Server или так:

```bash
npx serve src
```

После запуска открой локальный адрес из терминала.
